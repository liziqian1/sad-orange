## 主角设定

于文博 男 年龄不明

超能力是可以无限制地召唤砂糖橘。

可以使砂糖橘性质改变，
使碰到的人窜稀一周。

召唤时有概率出现炫酷五彩镭射炫光砂糖橘
（概率约为0.1％，是腾讯概率）

有保底，每召唤一万个砂糖橘必出一个五彩砂糖橘。

传说五彩砂糖橘有一种奇妙的力量。
